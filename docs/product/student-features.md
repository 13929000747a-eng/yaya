# 学生端功能指南 (Student Features)

## 1. 仪表盘 (Dashboard)

*   **路径**: `/dashboard`
*   **核心理念**: 极简主义 (Minimalist)，模仿 Notion 风格。
*   **主要功能**:
    *   **考季切换**: 顶部下拉菜单切换 "Jan-Apr 2026" 等考季。
    *   **进度追踪**: 还没做的题目显示 "New" (蓝点)，已完成的显示 "Studied" (灰点)。
    *   **分类展示**: 自动将题目分为 Part 1, Part 2, Part 3 三个标签页。

## 2. Topic Studio (学习工作台)

*   **路径**: `/topic/:id`
*   **布局**: 统一的左侧导航 + 右侧内容区。

### A. Part 1 练习模式
*   **交互**: 这是一个 Q&A 列表。
*   **查看答案**:以此点击 "View Answer" 展开高分示范答案。
*   **录音**: 每个问题下方都有独立的录音按钮。

### B. Part 2 练习模式 (Cue Card)
*   **界面**: 模拟真实考试的 Cue Card 卡片展示。
*   **Story Builder**:
    *   右侧提供 "Story Builder" 辅助工具。
    *   学生可以输入关键词 (Keywords) 构建故事线。
*   **计时器**: 1分钟准备倒计时 + 2分钟答题倒计时。

### C. Part 3 练习模式
*   **界面**: 深度讨论模式，问题通常关联 Part 2 的主题。
*   **逻辑训练**: 侧重于回答的逻辑框架 (观点 -> 解释 -> 举例 -> 结论)。

## 3. 个人中心 (Profile)

*   **路径**: `/profile`
*   **信息**: 包含昵称、目标分数、考试日期、家乡、专业等信息设置。
